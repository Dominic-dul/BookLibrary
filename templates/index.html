BOOK LIBRARY

Books

This page represents a book library with the list of books and certain action you can do with each book. These action are: add a new book, modify a book and delete a book. To add a new book, you have to provide book name, author and the year it was released. Similarly works the modification. You need to provide all the same fields and the book on the list is being updated. The id of the book is generated automatically.

Users

In this page you are able to see the list of users that are members of this book library. Furthermore, your are able to perform following actions with users: add a new user, delete a user and modify an existing user. To add a new user, you need to provide his name. The id is generated automatically. To modify a user, you can modify his name. To delete a user, you just need to press a button.

Lendings

In this page you can assign certain book from the library to a certain user. This is done by using drop-down menu and being able to see all the list of users and books

Games

This is a page for the game management part in the library. In this page you can perform these following actions: add a new game, delete a new game and modify a game. Adding and deleting a game works in the same way as with users and books. When modifying a book, you need to provide parts of the game by giving part name and the quantity of the game part.

This website was built using landing page template from w3.schools . Other tools used: html and css for front-end, postgresql for database and python back-end to connect database to the front-end
